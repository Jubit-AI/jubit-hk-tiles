# jubit-hk-tiles

**Hong Kong adaptation of [`cannoneyed/isometric-nyc`](https://github.com/cannoneyed/isometric-nyc).** Turns the Hong Kong Lands Department's territory-wide 3D open data into an isometric pixel-art tile library, plus curated game-ready backdrops.

```
HK Lands Dept 3D Open Data
        │
        ▼
 ┌──────────────────────────────────┐
 │ Forked render core (Three.js +   │
 │ orthographic camera) + Python    │
 │ bake pipeline (PyVista + Gemini  │
 │ + fine-tuned Qwen on Modal)      │
 └────────┬───────────────────┬─────┘
          │                   │
          ▼                   ▼
  Output A — Tile         Output B — ~12–20
  library (territory-     curated backdrops
  wide, DZI viewer) →     → jubuddy-HK game
  dseek.ai/data/life      assets
```

A separate, hand-authored isometric **game-grid tileset** (Output C, in the Jubit anime aesthetic) lives in the companion repo `jubuddy-hk` — not generated by this pipeline. See the [12-week plan](docs/HK-ADAPTATION-PLAN.md) for the why.

## Why fork (and what the upstream author warned us about)

The upstream author was explicit in their README: this codebase was built ~entirely via coding agents with <1% human review. We fork **eyes open**:

- The Three.js orthographic-camera renderer in `src/web_render/` is the part that just works. Reuse.
- The Modal/Oxen inference stack is solid for re-training Qwen on our aesthetic. Reuse.
- The NYC-specific data ingest (PostGIS schemas, NYC SRID, Google Maps satellite compositing) is **fully replaced** — HK data is richer (textured Cesium 3D Tiles direct from government, no satellite composite needed) so we skip the upstream's biggest false start.
- The water + vegetation cleanup tooling is critical for HK (Victoria Harbour and Country Parks are bigger surfaces than NYC's equivalents). Reuse + extend.
- Verticality occlusion (HK 400m towers next to 6-storey tong lau on slopes) is unsolved upstream. New R&D. See `docs/qa/verticality.md`.

## Legal compliance — non-negotiable

- **Attribution**: "Government of HK SAR — Lands Department" + "DATA.GOV.HK" on every consumer surface (the dseek.ai map + jubuddy-HK credits).
- **No third-party imagery composited in**: HK Cesium 3D Tiles only. No Google / Esri / Mapbox satellite overlays.
- **No tile resale**: HK terms prohibit data resale. We ship a *game* containing derivative assets (allowed) — not an asset pack (prohibited).
- **Trademark stylization**: Output must abstract IFC / ICC / Bank of China / MTR brand signage into unreadable color blocks.
- **MIT attribution** to `cannoneyed/isometric-nyc` retained.

Full text + checklist: [`docs/legal/attribution.md`](docs/legal/attribution.md).
