# HK Adaptation Plan — week-by-week

Tracks delivery of the 12-week plan approved 2026-06-13. The full plan lives at `~/.claude/plans/brainstorming-that-we-have-snoopy-adleman.md` on the operator's machine and is the source of truth; this file is the week-by-week status.

## Architecture (one-liner)

One shared deterministic Stage-1 render (Three.js orthographic camera over HK Cesium 3D Tiles) → two AI-styled outputs (full territory + curated backdrops) + one hand-authored output (game-grid tileset). See repo `README.md` for the visual.

## Outputs

| | What | Where it ships | Pipeline |
|---|---|---|---|
| **A** | HK Tile Library (~15–50K tiles, territory-wide) | `dseek.ai/data/life` pixel mode | This repo. Stage 1 + Stage 2A (Qwen fine-tuned on Jubit aesthetic). |
| **B** | Curated Backdrops (~12–20 hero locations) | `jubuddy-hk` game world map + event cards | This repo. Stage 1 + Stage 2B (curated subset of A with tighter QA). |
| **C** | Game-Grid Tileset (TD-legible) | `jubuddy-hk/apps/web/public/assets/stage-tiles/isometric/` | `jubuddy-hk` repo. Hand-authored, NOT generated by this pipeline. |

## Per-week deliverables

### Week 1 — Repo scaffold + docs

- ✅ Fork `cannoneyed/isometric-nyc` → `Jubit-AI/jubit-hk-tiles`
- ✅ Apply Jubit 12-rule template (`CLAUDE.md` — preserves upstream tech-stack guidance below)
- ✅ Replace `README.md` (preserve upstream as `README.upstream.md`)
- ✅ Add `docs/HK-ADAPTATION-PLAN.md` (this file)
- ✅ Add `docs/qa/verticality.md` — HK-specific R&D doc
- ✅ Add `docs/legal/attribution.md` — HK terms + MIT inheritance
- ✅ Add `scripts/request-hk-api-key.md` — email template (operator action)
- ✅ Add `src/isometric_nyc/data/hk_lands_dept.py` — stubbed HK Cesium 3D Tiles adapter, ready for the API key

### Week 2 — Central pilot render *(blocked on HK API key)*

- ⬜ Operator sends API key request (`scripts/request-hk-api-key.md`)
- ⬜ Wire `hk_lands_dept.py` to fetch Central tileset.json
- ⬜ Run Stage 1 (deterministic Three.js orthographic render) on Central tiles
- ⬜ **Verification gate**: visually inspect ~20 Central tiles. PASS = tiles look acceptable; FAIL = re-tune camera or accept stylistic occlusion. See `docs/qa/verticality.md`.

### Week 3–4 — Jubit aesthetic fine-tune

- ⬜ Build 40-pair training set in Jubit anime palette (the design anchor — without this we inherit SimCity 2000)
- ⬜ Fine-tune Qwen/Image-Edit on Modal (~$12, ~4 hrs per upstream's published numbers)
- ⬜ Generate 100 tiles with AI infill
- ⬜ **Verification gate**: user reviews 100-tile sample. PASS = aesthetic harmonizes with Jubit character art; FAIL = re-curate training set, re-train.

### Week 5–6 — `jubuddy-hk` scaffold + hand-authored Output C

- ⬜ Scaffold `jubuddy-hk` from `jubuddy-game` template (separate repo)
- ⬜ Add isometric renderer alongside existing top-down renderer in `apps/web`
- ⬜ Hand-author game-grid tileset in Jubit anime aesthetic (`path` / `slot` / `blocked` / `spawn` / `goal`)
- ⬜ First 3–5 playable maps using Output C
- ⬜ **Verification gate**: jubuddy-HK alpha with 3 maps. PASS = end-to-end playable, no renderer crashes, pathfinding correct on isometric grid.

### Week 7–10 — Full-territory generation (Output A)

- ⬜ 50× Modal parallel
- ⬜ Iterative QA on water, vegetation, verticality (district batches: Central → TST → Causeway Bay → Kowloon → NT → outlying islands)
- ⬜ DZI export
- ⬜ Deploy OpenSeaDragon viewer to `dseek.ai/data/life`
- ⬜ **Verification gate**: random sample of 50 tiles reviewed. PASS = <5% need manual rework; FAIL = systemic issue, re-train.
- ⬜ Budget watch: $1–2k Modal/Gemini spend. Manual QA time is the hidden cost.

### Week 11 — Curated backdrops (Output B) + legal spot-check

- ⬜ Re-run Stage 2A on ~20 hero locations (Victoria Harbour, Peak, Kowloon canyon, MTR station, outlying island, etc.)
- ⬜ Integrate into jubuddy-HK as world-map skins + event-card backgrounds
- ⬜ **Verification gate (legal)**: visual sweep for legible trademarks/signage. PASS = no readable corporate brand text. FAIL = re-stylize or re-frame.

### Week 12 — Launch

- ⬜ Attribution screens on both consumer surfaces
- ⬜ Performance test dseek.ai/data/life zoom UX (60 fps gate)
- ⬜ jubuddy-HK alpha
- ⬜ Bi-annual auto-refresh follow-up filed as deferred

## Cost envelope

- Fine-tune: ~$12
- Full-territory generation: ~$1–2k one-shot (peak weeks 7–10)
- Hosting (DZI on CDN): negligible
- **Hidden cost**: manual QA. Upstream author says "the last 10% took 90% of the time."
